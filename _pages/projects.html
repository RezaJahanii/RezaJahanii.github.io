Wireless System Simulation
In this project, according to different types of fading, a wireless system is simulated and some parameters such as received power, outage probability, coverage area of the base station, etc are calculated.
(Instructor: Dr. Maryam Sabbaghian)

Narrow-Band Wireless Channel Simulation
Using different modulations or time diversity and space diversity, a wireless system is simulated and system performance in each case is analyzed. (Instructor: Dr. Maryam Sabbaghian)

OFDM System Implementation
A message with certain length is transmitted in a frequency selective channel via OFDM system. For this purpose, and OFDM system is designed and implemented. System performance with modules of Water-Filling, ZF & MMSE Equalizers, and ... is analyzed. (Instructor: Dr. Maryam Sabbaghian)

Real Time Object Detection with YOLOv3
Employing YOLOv3, a model is developed to detect 3 classes of Bus, Person, and car real time. For evaluation of the model, a test video is fed to the network and the output video has bounding boxes around the desired objects in the video. (Instructor: Dr. Reshad Hosseini)

Facial Emotion Recognition using Deep Learning Methods
Using FER2013 dataset, a CNN based deep learning method is proposed for intelligent facial emotion classification. Simulations illustrate significant improvement in comparison to baseline models. (Instructor: Dr. Edgar Lobaton)

Surface Detection using Deep Learning
A deep learning framework is developed for classification of the surface the subject is walking on using a 6 channel signal obtained from accelerometers and gyroscopes. (Instructor: Dr. Edgar Lobaton)

Developing Metric Based KNN Classifier
Using LMNN and LFDA after PCA to change the space of the datas and then implementing a KNN classifier on the mapped data. (Instructor: Dr. Reshad Hosseini)

Implementing a Deep Learning Model on CIFAR-10 Dataset
In this project, a neural network structure is developed for the purpose of classifying CIFAR-10 labels. Different hyperparameters are tuned and the optimum neural network for the classification is reached.
(Instructor: Dr. Reshad Hosseini)

Naive Bayes Model Development on MINST Dataset
Initially, features are extracted from the dataset. Furthermore, a naive bayes model is developed for classification of MINST dataset. (Instructor: Dr. Reshad Hosseini)

Q-Learning Model Free Reinforcement Learning
In this project, an agent is trained for taxi-v3 game employing Q-Learning approach. The agent's intention is to pick up the fairs and dropping them at their destination. (Instructor: Dr. Reshad Hosseini)

Deep Learning Based DOA Estimation
A deep learning model based on convolutional neural networks is designed and simulated to predict direction of arrival. This model outperforms conventional methods such as MUSIC, Beamforming, etc.
(Instructor: Dr. Saeed Akhavan Behabadi)

BCI Signal Processing
A model based on CSP filter and LDA classifier is implemented to classify the action subject would take according to the obtained EEG signal from brain. (Instructor: Dr. Saeed Akhavan Behabadi)

Implementation of Sparse Recovery Methods
Several methods for sparse recovery problem are implemented in this project and evaluated.
(Instructor: Dr. Saeed Akhavan Behabadi)

Independent Component Analysis
Several methods for independent component analysis are simulated in this project for the purpose of blind source separation problem in case of independent sources. (Instructor: Dr. Saeed Akhavan Behabadi)

Dictionary Learning with K-SVD & MOD
Implementation of dictionary learning methods, K-SVD and MOD, for blind source separation problem in case of presence of sparse sources. (Instructor: Dr. Saeed Akhavan Behabadi)

Movement Classification with Common Spatial Pattern
A model is developed based on CSP filters to map multi-channel T sampled observations to a feature vector for purpose of classification between arm movement and leg movement. (Instructor: Dr. Saeed Akhavan Behabadi)

Canonical Correlation Analysis for Frequency Detection in SSVEP
A CCA model is developed for prediction of the frequency in steady state visually evoked potentials.
(Instructor: Dr. Saeed Akhavan Behabadi)

Implementation of Digital Communication System with 4PAM
Assuming an AWGN channel, a digital communication system is simulated which employs 4-PAM for transmitting the symbols. Each step at the transmitter and receiver is implemented in this project including generation of pulses, etc. (Instructor: Dr. Amir Masoud Rabeie)

Compressing Image Employing 2D Fourier Transform
In this section of the project, it is aimed to compress the images via 2D fourier transform.
(Instructor: Dr. Majid Badieirostami)

Mind State & Behaviour Prediction with EEG Brain Signal Analysis
By performing signal processing methods on EEG signals taken from subjects, such as separating low frequency and high frequency parts of the signal, state of one's mind is predicted. (Instructor: Dr. Majid Badieirostami)

Audio Processing using Filters
In this project, a voice which has echo and noise goes under process and using signal processing methods and filters, echo is deleted and noise is reduced. (Instructor: Dr. Majid Badieirostami)

Yagi-Yuda Antenna Design
A Yagi-Yuda antenna was designed and created with gain of 3.43dB and bandwidth of 500MHz in antenna lab using HFSS.
